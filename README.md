## วิธีลบไฟล์ใน Termux (remove)
- ``` rm ``` ตามด้วยชื่อไฟล์
- ```rm -r```  ตามด้วยชื่อโฟลเดอร์
- ```rm -rf``` เพื่อลบโฟลเดอร์พร้อมเนื้อหาแบบบังคับโดยไม่ถามยืนยัน
